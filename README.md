
# Football Quant ⚽📊

**双变量泊松分布 + Elo动态评分 + 贝叶斯凯利仓位管理的足球量化分析框架**

---

## 为什么做这个？

市面上大部分足球预测工具，要么是"猜胜负"的噱头，要么是纯数据拟合就交差了。

这套系统的核心假设：**比赛结果不可精确预测，但概率可以建模。**

我的背景是量化交易（20年），这套模型是把交易中的风险管理理念迁移到足球赛事分析——关注的不是"这场谁赢"，而是**长期能不能找到正期望值的下注机会**。一个策略有没有用，不是看它猜对了几场，是看回测的Z-score和ROI能不能站住。

## 架构

```
原始数据（football-data.co.uk）
        │
        ▼
  数据加载 + xG匹配 ────→ DC双变量泊松（进球分布建模）
        │                          │
        │                    Elo动态评分（球队实力追踪）
        │                          │
        │                   概率融合（DC + Elo）
        │                          │
        │                  贝叶斯凯利（仓位管理）
        │                          │
        ▼                         ▼
  回测引擎 ────→ 蒙特卡洛模拟 ────→ Z-score / ROI 评估
```

### 核心模块

| 模块 | 文件 | 作用 |
|------|------|------|
| 配置 | `config.py` | 所有参数统一管理，方便调参 |
| 数据 | `src/data_loader.py` | 读取CSV、过滤Top10球队、匹配xG |
| 模型 | `src/models.py` | DC双变量泊松、Elo、贝叶斯评级 |
| 策略 | `src/strategy.py` | 概率融合、贝叶斯凯利仓位、滑点处理 |
| 回测 | `src/backtest.py` | 回测引擎、Z-score、蒙特卡洛模拟 |
| 入口 | `run.py` | 一键运行 |

## 快速开始

```bash
# 1. 克隆
git clone https://github.com/tyj-da/football-quant.git
cd football-quant

# 2. 运行（Python 3.10+，只需标准库，无需额外包）
python run.py
```

数据已包含在仓库中（五大联赛×4赛季 + xG缓存），克隆完直接跑。

### 数据格式

CSV需包含以下字段：
- `Date` — 比赛日期（DD/MM/YY 或 YYYY-MM-DD）
- `HomeTeam`, `AwayTeam` — 主客队名
- `FTHG`, `FTAG` — 全场主客队进球数
- `PSH`, `PSD`, `PSA` — 终盘主胜/平/客胜赔率

## 回测结果（真实数据）

基于五大联赛×4赛季（2223~2526）共1583场Top10球队比赛，包含xG数据。

```
  ==================================================
  回测结果 (1670 次押注)
  ==================================================
    胜率:        25.1%
    ROI:         +10.75%
    Z-score:     0.452
    总押注额:    19.60
    总盈亏:      +2.11

  蒙特卡洛模拟 (0/1000 次破产)
    平均ROI:     -8.4%
    中位数ROI:   -59.1%

  按联赛拆分
    意甲        269注  ROI:+23.17%  Z:+0.36
    西甲        266注  ROI:-4.28%   Z:-0.08
    德甲        260注  ROI:+11.08%  Z:+0.19
    英超        256注  ROI:+4.85%   Z:-1.44
    法甲        176注  ROI:+64.19%  Z:+2.05
```

**⚠️ 注意：** 除法甲（Z=2.05）外，其余联赛的Z-score均未达到统计显著性阈值（|Z|>2），当前结果有相当可能性来自随机波动。蒙特卡洛模拟的中位数ROI为负值，说明策略的波动较大。**这不是一个已验证可稳定盈利的策略，而是一个仍在迭代中的量化实验。**

**关键指标：**
- **ROI**：总回报率。正的说明策略赚钱，负的说明亏钱
- **Z-score**：衡量"这个结果是不是运气"。通常 |Z| > 2 认为有统计显著性
- **蒙特卡洛破产率**：随机波动下亏光的概率，越低越稳健
- **按联赛拆分**：看看策略在哪几个联赛有效，在哪几个无效

## 数据来源

比赛数据来自 [football-data.co.uk](https://www.football-data.co.uk/)，免费提供历史赛果和赔率数据。xG 数据从公开API获取。
